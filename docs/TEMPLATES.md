# Templates

This document outlines the base folder structure templates for 2D/3D Unity projects with one simple and one comprehensive configs for each.

</div>

## 2D Project Structures

### 2D Folder Config

```text
_Project/
├── Art/
│   ├── Animations/
│   ├── Materials/
│   ├── Shaders/
│   ├── Sprites/
│   ├── Tilesets/
│   └── Fonts/
├── Audio/
│   ├── Music/
│   └── SFX/
├── Prefabs/
│   ├── Characters/
│   ├── Environment/
│   └── UI/
├── Scenes/
│   ├── Gameplay/
│   ├── Menus/
│   └── Test/
├── Scripts/
│   ├── Core/
│   ├── Gameplay/
│   ├── UI/
│   └── Data/
├── Settings/
│   ├── Input/
│   └── Rendering/
├── Tests/
└── Documentation/
Plugins/
StreamingAssets/
Editor/
Resources/
ThirdParty/
```
